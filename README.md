# wc — клон утилиты Unix `wc`

Реализация утилиты `wc` (word count) на языке C.

Считает в файле:
- `-l` — количество строк
- `-w` — количество слов
- `-c` — количество байт (символов)

Если флаги не указаны, выводит все три значения.
Если файлов передано больше одного, дополнительно выводится строка `total`.

## Сборка

```bash
make
```

Собирается через GCC с флагами `-Wall -Wextra -Werror -std=c11 -O2`.

## Использование

```bash
./wc file.txt              # выведет все три счётчика
./wc -l file.txt           # только строки
./wc -w file.txt           # только слова
./wc -c file.txt           # только байты
./wc -lw file.txt          # строки и слова
./wc a.txt b.txt           # каждый файл + строка total
```

## Тесты

```bash
make test
```

Тесты сравнивают вывод этой реализации с системным `wc`.

## Архитектура `src/wc.c`

Программа разбита на маленькие функции с одной ответственностью:

| Функция | Что делает |
|---|---|
| `options_init`, `counts_init` | Инициализируют структуры значениями по умолчанию |
| `count_file` | Считает строки/слова/байты в одном открытом файле через посимвольный проход |
| `parse_flags` | Разбирает аргументы командной строки и заполняет `options_t` |
| `process_one_file` | Открывает файл, вызывает `count_file`, печатает результат, обрабатывает ошибки |
| `add_counts` | Прибавляет счётчики одного файла к итогу |
| `main` | Связывает всё вместе |

Подсчёт слов реализован через **конечный автомат** на двух состояниях (`in_word = 0/1`), что позволяет корректно обрабатывать любые комбинации пробельных символов между словами.
